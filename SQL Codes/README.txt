Hello everyone, I am happy to throw a project on github for the first time in this project. First, my goal is to create a G-value to measure the value of Call Center and Customer Satisfaction by putting the dialogs recorded as text files into a formula by myself. These values are values that indicate which Call Center element is successful and to what extent the customer is satisfied. The F values that are formed in the project show the success measurements of the dialogues. The functioning of the mechanism is given in the PowerPoint Presentation. Turkish englishmen are the most popular Turkish language.Finally, I have prepared a mechanism that transfers a text spoken in Turkish to a database by translating it into English with the help of a microphone to add a little reality.
